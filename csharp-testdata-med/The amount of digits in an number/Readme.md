# The amount of digits of a number
A simple console application that accepts a number from a user and outputs the number of digits of an entered number.

## Features
- Accepts entered integer from a user
- Displays the number of digits in a number

## Getting Started
1. Clone or download the repository to your local machine.
2. Open the solution in your preferred C# development environment.
3. Build the solution to compile the code.
4. Run the application.

## Usage
1. Upon running the application, you will be prompted to enter a number.
2. Enter the number for which you want to find the number of digits.
3. The application will display the number of digits of the entered number.
